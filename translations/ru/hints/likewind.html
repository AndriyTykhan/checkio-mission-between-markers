        begin_index = text.find(begin) + len(begin)

Сперва мы проверяем, есть ли левый маркер в тексте. Если его там нет - то ответ будет начинаться с самого начала исходной строки. Если он там есть, то ответ будет начинаться с символа, следующего за последним символом маркера. Именно поэтому в коде к позиции самого маркера - 'text.find(begin)', добавляется еще и его длина - '+ len(begin)'.
А что делать с правым маркером?
Ситуация аналогичная, только в случае, если правого маркера нет в исходном тексте - за конец строки мы принимаем конец исходного текста. Вот так: 
if end in text:
    end_index = text.find(end)
else:
    end_index = len(text)
Как получить текст, находящийся между маркерами?
Используя срез (slice): 
        
answer = text[begin_index:end_index]
Как объединить это вместе?
Полный код может выглядеть так: 
def between_markers(text: str, begin: str, end: str) -> str:
    if begin in text:
        begin_index = text.find(begin) + len(begin)
    else:
        begin_index = 0
    
    if end in text:
        end_index = text.find(end)
    else:
        end_index = len(text)
        
    return text[begin_index:end_index]
